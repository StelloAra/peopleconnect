# PeopleConnect

Ett modernt skolprojekt byggt med **Next.js**, **TypeScript** och **Firebase**.

Projektet fokuserar på:

- Inloggning med Firebase Authentication
- Grundläggande användarflöde
- Modern webbstruktur med Next.js

## Tekniker

- Next.js
- TypeScript
- Firebase (Authentication, Firestore)

## Starta projektet lokalt

```bash
npm install
npm run dev
```
